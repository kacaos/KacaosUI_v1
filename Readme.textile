p. +What is *{color:deepskyblue}KacaosUI*?+

p. *{color:deepskyblue}KacaosUI* is an external Addon which skins the original *{color:darkorchid}Tukui*.

p. This ui have an minimalistic look and its an extreme lightweight user interface for World of Warcraft.

p. Best resolution is 1920x1080, because thats mine. Lower resolutions should work too, i hope so !http://www.wowinterface.com/images/style_mmoi_wowi2/smilies/wink.gif!

p. +Required+

p. "*{color:darkorchid}Tukui*":https://github.com/tukz/Tukui/downloads is <i>required</i> to run this addon!
